# Next.js app (minimal)

This folder contains a minimal Next.js app that uses the Supabase client to read farmers.

Setup:
1. Copy .env.example -> .env.local and fill SUPABASE_URL and SUPABASE_ANON_KEY
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`

The page /farmers will list farmers from your Supabase DB.
